# Qwiic Cable and Connector Specification

- SparkFun created Qwiic
- It simplifies sensor connections.
- Communication uses I²C.
- Supply voltage: 3.3 V.
- Logic level: 3.3 V.
- Not normally 5 V tolerant.
- Connector: JST-SH.
- Four contacts.
- Pitch: 1.0 mm.
- Pins: GND, 3.3 V, SDA, SCL
